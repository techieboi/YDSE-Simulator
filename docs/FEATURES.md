# Feature List

## Core Product Features

- Real-time double slit interference simulation
- Physics parameter controls
- Preset demonstrations for common light conditions
- Derived values panel for fringe spacing and path difference
- Educational explanation cards
- Responsive browser layout

## MVP Features

- Interference screen visualization
- Intensity profile graph
- Controls for:
  - Wavelength
  - Slit separation
  - Slit width
  - Screen distance
  - Brightness
- Preset buttons
- Physics formulas panel
- Quick explanation of bright and dark fringes

## User-Accessible Features

- Move sliders and see the pattern update live
- Select presets such as red light, blue light, and wider slit gap
- Read the current fringe spacing and screen interpretation
- Use the simulator in any modern browser window
- Present from desktop or projector without extra setup

## Post-MVP Ideas

- Animated wavefront view
- Constructive vs destructive interference markers
- Single slit comparison mode
- Presentation notes mode
- Export screenshot button
