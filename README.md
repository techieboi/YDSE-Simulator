# Interactive Double Slit Interference Simulator

A browser-based physics presentation project built with React, Vite, and Tailwind CSS.
It provides a formula-accurate simulation for Young's Double Slit Experiment (YDSE).
The user can change the control paramteres in real-time and the projection canvas along with the intensity graph will reflect those changes.

## Getting Started
Fork the repository and run locally-
```bash
npm.cmd install
npm.cmd run dev
```

## Scripts

- `npm.cmd run dev` starts the development server.
- `npm.cmd run build` creates a production build.
- `npm.cmd run lint` runs ESLint.
