# Interactive Double Slit Interference Simulator

A browser-based physics presentation project built with React, Vite, and Tailwind CSS.

## Getting Started

```bash
npm.cmd install
npm.cmd run dev
```

## Scripts

- `npm.cmd run dev` starts the development server.
- `npm.cmd run build` creates a production build.
- `npm.cmd run lint` runs ESLint.
